# Система Хранения Электронных Документов
## Описание
 Этот проект представляет собой программный модуль системы хранения электронных документов, предназначенный для интеграции с системой электронного документооборота фирмы,
занимающейся техническим обслуживанием торгового оборудования. Модуль предоставляет функциональность для пользователей и модераторов (администраторов), обеспечивая удобный
доступ к документам и управление ими.
## Функциональность для Пользователей
- Просмотр информации о приеме электронных дел, документов и описей структурных подразделений с проверкой комплектности.
- Получение оповещений о выполнении проверки электронных подписей документов.
- Просмотр истории взаимодействий и переписки по существующим архивам со структурными подразделениями.
- Прикрепление файлов к заявкам.

## Функциональность для Модераторов (Администраторов)
- Добавление пользователей сервиса в базу.
- Просмотр информации о регистрации электронных дел, документов и описей структурных подразделений с проверкой комплектности.
- Определение статусов заявок, приоритетов заявок и типов заявок на регистрацию.
